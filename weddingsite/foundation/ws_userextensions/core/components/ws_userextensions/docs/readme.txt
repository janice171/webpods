
WS_UserExtensions component for MODx Revolution

Purpose: Base user package for the wedding sites project
Author: S. Hamblett <steve.hamblett@linux.com>
For: http://www.weddingwebsitesuk.co.uk/ 9thWave New Media
Date: 10/05/2011

This package extends the MODX Revolution user class and creates
the user extension tables necessary for the wedding site. As such it
must be installed as a pre-requisite to other wedding site packages.
